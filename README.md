## Featured Work

- **FedSpeak** — NLP pipeline on 240 FOMC meeting minutes (2000–2026), detecting structural linguistic shifts around the 2008 financial crisis
- **Double ML: 401(k)** — Causal effect of 401(k) eligibility on net financial assets using cross-fitted Random Forest learners across ~10,000 observations
- **Fraud Detection** — Demonstrated the accuracy paradox on a 0.172% positive-rate dataset and optimized decision thresholds under operational constraints
- **World Economy Clustering** — Unsupervised grouping of 160 economies using World Bank indicators, recovering World Bank income tier structure at K=4
